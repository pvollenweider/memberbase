Documentation

Tout ce qu’il faut pour utiliser, installer et étendre MemberBase. Quatre guides, du quotidien d’un utilisateur jusqu’aux entrailles du code.

  • Guide utilisateur

    Pour toute personne qui utilise MemberBase au quotidien.

    Connexion, liste des membres, fiches, comptabilité et suivi, import de contacts CSV, segments, attestations de dons, exports, rôles et droits.

  • Guide administrateur

    Pour la personne qui installe et maintient l’application.

    Prérequis, installation par wizard web ou Docker, déploiement Apache, gestion des comptes, sécurité, sauvegarde, migrations de schéma et logs d’audit.

  • API REST

    Pour intégrer MemberBase à d’autres outils.

    Endpoints JSON pour membres, segments, comptabilité et suivi : authentification, rôles, paramètres, exemples curl et format des réponses.

  • Architecture

    Pour comprendre ou modifier le code.

    Flux de requête htmx, couches applicatives, classes de domaine, schéma de base de données, frontend, tests et conventions de code.

  • FAQ

    Pour trouver une réponse rapide à une question courante.

    Cas pratiques par persona : membres, segments combinés, cotisations, dons, import, export, administration, API et contribution au projet.
